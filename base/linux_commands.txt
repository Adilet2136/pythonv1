ls-показывает все файлы и папкм в которой в нахоходитсь
ls -a -показывает сфайлы и деректорий а также скрытые файлы и деректорий
ls -l -показывает более подробную информацию о файлах и папках
ls -la - показывает дополнительную информацию о файлах и папках + скрытых тоже
ls путь до папки - показывает содержимое папкм не зоходя туда через путь
ls Desktop/pythonv1/base - пример
cd -переход в домшниюю папку
cd ~ - переход в домашнию папку
cd name - переходит в папку name если такая папка существует в текущей дириктории
cd . - в текущию  папку
cd .. переходит на одну папку назад 
cd ../.. -переходит на две папки назад
mkdir test - создает дирикторию test в текущию папке
mkdir путь/test - создает дирикторию test по пути
mkdir test1 test2 test3 -создаст не сколько папок  test1 test2 test3 в текущей папке
touch test.txt -  создание файла test с раширениeм txt
rm test - удаляет файл
rmdir test - удаляем папку если она пустая
rm -rf test - удаляет папку test пустая или не пустая не имеет значениия
mv test100.txt test.txt - переминовыает  файлы и папки
mv test1.txt dirtest - перемистит файл test.txt в папку dirtest
history - показывает которые вводили в данном терминале
history 10 - показывает последние 10 команд в данном терминале
pwd - показывает путь до той деректорий где вы выполнили эту команду
nano test.txt - откроет файл test.txt в текстовом редакторе nano если такого файла не то создаст его
cat test.txt - выведит в терминале содержимое test.txt 
apt - устоновчик пакетов 
snap- системный устоновчик пакетов
sudo -super user do
